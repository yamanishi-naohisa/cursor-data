## Cursor Webとは

Cursor Webは、ブラウザ上で動作するAI支援のコードエディタです。デスクトップ版と同様の機能を、インストール不要で利用できます。

### 主な特徴
- ✅ **インストール不要**: ブラウザから直接アクセス可能
- ✅ **クロスプラットフォーム**: Windows、macOS、Linux、ChromeOSなど対応
- ✅ **AI機能搭載**: コード補完、生成、レビュー機能
- ✅ **クラウド同期**: デバイス間でプロジェクトを共有可能
- ✅ **自動更新**: 常に最新バージョンが利用可能

---

## アクセス方法

### 1. システム要件
- **対応ブラウザ**: 
  - Chrome 88以上
  - Edge 88以上
  - Firefox 86以上
  - Safari 14以上
- **インターネット接続**: 安定したブロードバンド接続（5Mbps以上推奨）
- **Cursorアカウント**: 利用にはアカウント登録が必要

### 2. ログイン手順
1. ブラウザで [https://web.cursor.sh](https://web.cursor.sh) にアクセス
2. Cursorアカウントでサインイン
3. 初回利用時は、オンボーディングプロセスを完了

## スマートフォンからの使用方法

### 概要

Cursor Webはスマートフォンからも利用可能ですが、画面サイズの制約により、一部の機能は制限されます。主にコードの確認、簡単な編集、AIチャットでの質問などに適しています。

### 対応デバイスとブラウザ

#### iOS（iPhone/iPad）
- **Safari**: iOS 14以上推奨
- **Chrome**: iOS版Chrome（最新版）
- **Firefox**: iOS版Firefox（最新版）

#### Android
- **Chrome**: Android 8.0以上推奨
- **Firefox**: Android版Firefox（最新版）
- **Edge**: Android版Edge（最新版）

### アクセス方法

#### 1. ブラウザでアクセス
1. スマートフォンのブラウザを開く
2. アドレスバーに `https://web.cursor.sh` を入力
3. または、ブックマークに保存して簡単にアクセス

#### 2. ホーム画面に追加（PWA対応）
**iOS（Safari）の場合:**
1. Safariで [https://web.cursor.sh](https://web.cursor.sh) を開く
2. 画面下部の「共有」ボタン（□↑）をタップ
3. 「ホーム画面に追加」を選択
4. 名前を確認して「追加」をタップ

**Android（Chrome）の場合:**
1. Chromeで [https://web.cursor.sh](https://web.cursor.sh) を開く
2. メニュー（⋮）をタップ
3. 「ホーム画面に追加」を選択
4. 確認して「追加」をタップ

### 基本操作（スマートフォン版）

#### 1. プロジェクトを開く
1. メニューアイコン（☰）をタップ
2. 「プロジェクトを開く」を選択
3. 最近のプロジェクトから選択、または新規作成

#### 2. ファイルの閲覧・編集
1. ファイル一覧からファイルをタップ
2. エディタ画面でコードを表示
3. コードをタップして編集モードに切り替え
4. キーボードでコードを編集

1. 画面下部の「💬 AIチャット」ボタンをタップ
2. チャット画面が開く

4. 送信ボタンをタップ

1. コードを長押しして選択開始

3. コンテキストメニューから「コピー」を選択

### スマートフォン版の制限事項

#### 機能的な制限
- ❌ **ショートカットキー**: 物理キーボードがないため、一部のショートカットが使用不可
- ❌ **複数ファイルの同時編集**: 画面サイズの制約により、タブ表示が制限される
- ❌ **ターミナル操作**: フル機能のターミナルは利用しにくい
- ⚠️ **コード補完**: 入力は可能だが、デスクトップ版ほど快適ではない
- ⚠️ **インライン編集**: コード選択と編集指示の入力がやや煩雑

#### 推奨される用途
- ✅ **コードの確認**: 外出先でコードを確認
- ✅ **AIチャットでの質問**: プログラミングに関する質問
- ✅ **簡単な編集**: 小さな修正やコメントの追加
- ✅ **コードレビュー**: AIにコードをレビューしてもらう
- ✅ **アイデアのメモ**: 思いついたアイデアをAIチャットで記録

### スマートフォン版の操作のコツ

#### 1. 横向きモードの活用
- スマートフォンを横向きにすると、コードが見やすくなります
- 特に長いコード行の編集時に有効

#### 2. ズーム機能の使用
- ピンチアウトでコードを拡大
- 細かい編集時に便利

#### 3. 外部キーボードの接続
- Bluetoothキーボードを接続すると、入力が快適になります
- ショートカットキーも一部使用可能

#### 4. 音声入力の活用
- AIチャットでの質問に音声入力を使用
- 長文の入力が楽になります

### トラブルシューティング（スマートフォン版）

#### よくある問題

1. **画面が正しく表示されない**
   - ブラウザを再起動
   - キャッシュをクリア
   - 別のブラウザで試す

2. **タッチ操作が反応しない**
   - ページをリロード
   - ズームレベルをリセット（ピンチイン/アウト）

3. **キーボードが表示されない**
   - 入力欄を再度タップ
   - ブラウザの設定を確認

4. **AIチャットが開かない**
   - インターネット接続を確認
   - ページをリロード
   - ブラウザを最新版に更新

5. **コードが保存されない**
   - 自動保存設定を確認
   - 手動で保存ボタンをタップ
   - ブラウザのストレージ容量を確認

### スマートフォン版のベストプラクティス

1. **Wi-Fi接続の推奨**
   - モバイルデータよりも安定した接続
   - データ使用量を節約

2. **バッテリー管理**
   - 長時間使用時は充電器を接続
   - 画面の明るさを調整

3. **セキュリティ**
   - 公共のWi-FiではVPNを使用
   - ログアウトを忘れずに

4. **パフォーマンス**
   - 不要なタブを閉じる
   - ブラウザのキャッシュを定期的にクリア

## 主要機能

### 1. AIチャット機能

**使い方:**
- `Ctrl + L`（Mac: `Cmd + L`）でチャットパネルを開く
- 画面右側にAIチャットウィンドウが表示される
- 質問や指示を入力すると、AIがコードの生成や修正をサポート

**使用例:**
```
ユーザー: "Pythonでリストをソートする関数を作成してください"
AI: [コードを生成]
```

### 2. インライン編集

**使い方:**
1. 修正したいコードを選択
2. `Ctrl + K`（Mac: `Cmd + K`）を押す
3. プロンプトに修正内容を入力
4. `Ctrl + Y`（Mac: `Cmd + Y`）で適用

**使用例:**
```
選択したコード: def hello():
                    print("Hello")

Ctrl + K → "関数名をgreetに変更し、引数nameを追加"
→ AIが修正案を提示
Ctrl + Y → 適用
```

### 3. コード自動補完

**使い方:**
- コード入力中にAIが補完候補を自動表示
- `Tab`キーで補完を確定
- 複数の候補がある場合は矢印キーで選択

### 4. ドキュメント参照

**使い方:**
- `@docs`と入力した後にライブラリ名を入力
- AIが外部ライブラリやフレームワークの公式ドキュメントを参照

**使用例:**
```
@docs React
→ Reactの公式ドキュメントを参照して回答
```

### 5. コードレビュー

**使い方:**
- コードを選択してAIチャットで「このコードをレビューしてください」と入力
- AIが改善点やバグを指摘

   - ブラウザのコンソールでエラーを確認

3. **コード補完が表示されない**
   - 設定でAI補完が有効か確認
   - ファイルの言語が正しく認識されているか確認
